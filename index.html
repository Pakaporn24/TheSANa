1. เป็นเว็บในการเลือกซื้อเสื้อผ้า

3. ในแต่ละสินค้าจะมี ราคา และ Code ถ้าต้องการซื้อสิ้นค้าให้กดไปที่ Buy จะขึึ้นแถบให้ใส่ Code และจำนวนสิ้นค้าที่จะซื้อ และกดสั่งซื้อ

4. ในหน้า Home จะมีแถบสำหรับให้ลงชื่อเพื่ือลุ้นรับของรางวัล
